# About CapyHome

> **Super Agent Harness — Batteries Included, Fully Extensible**

**CapyHome** is an open-source super agent harness that orchestrates sub-agents, memory, and sandboxes to perform complex, multi-step tasks — powered by extensible skills.

---

## Core Features

* **Skills & Tools**: 18 built-in skills with progressive loading. Add custom skills or extend via MCP servers.
* **Sub-Agents**: Parallel task delegation with scoped context and automatic synthesis.
* **Sandbox & File System**: Isolated execution environments (Local, Docker, Kubernetes).
* **Context Engineering**: Isolated sub-agent context, automatic summarization.
* **Long-Term Memory**: Persistent fact extraction across sessions.
* **IM Channels**: Telegram and Slack integration.
